Normativas y Buenas Prácticas

La accesibilidad web se rige por las WCAG (Web Content Accessibility Guidelines), un conjunto de pautas internacionales que aseguran que cualquier persona pueda usar una web, incluyendo usuarios con discapacidad visual, auditiva, motriz o cognitiva.

En España y la Unión Europea, la accesibilidad está regulada por la Ley 34/2002, el RD 1112/2018 y la Directiva Europea 2016/2102, que obligan a que las páginas públicas y muchas privadas cumplan al menos el nivel AA de las WCAG.

Niveles de Conformidad WCAG

Nivel Descripción
A Requisitos mínimos indispensables para que un sitio sea usable con tecnología asistiva.
AA Equilibrio entre accesibilidad y diseño. Nivel exigido por la normativa europea.
AAA Máximo nivel de accesibilidad. Recomendado para proyectos inclusivos avanzados.

Buenas Prácticas en Accesibilidad Web

  • Usar contraste suficiente entre texto y fondo.
  • Añadir textos alternativos en imágenes.
  • Estructurar el contenido con encabezados claros (H1, H2, H3...).
  • Garantizar navegación por teclado.
  • Evitar animaciones rápidas o parpadeos.
  • Proveer transcripciones o subtítulos para vídeos.

Comparativa Visual

Diseño No Accesible

Bajo contraste, texto pequeño, mala estructura y falta de etiquetas descriptivas.

Diseño Accesible

Contraste adecuado, jerarquía visual clara, accesibilidad para teclado y lectores de pantalla.
